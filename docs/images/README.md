## Recommended Conventions for Future Screenshots

- Use PNG format.
- Prefer widths around 1400px when capturing full UI screens.
- Keep screenshots readable on GitHub when scaled to ~950–1100px in the README.
- Use numbered filenames for ordered walkthroughs.
- Keep example flows grouped under `docs/images/examples/<scenario-name>/`.
